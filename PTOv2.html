PTO V2.0

A Program that calculates the PTO(Paid time off) for the corresponding amount of time the employee has work years. returns amount of PTO in weeks.

you earn 5hrs perpaycheck. We get 26 paychecks a year.

you earn hours per year.

  1. START
  2. SET INT hrcPerPayCheck
  3. SET INT paychecksPerYear = 26
  4. INPUT yearsWorked
  5. if yearsWork less than 5 hrsPerPaycheck
  6. if yearsWorked equal to or greater than 5 hrsPerPaychech = 6
  7. if yearsWorked equal to or greater than 10 hrsPerPaycheck = 7
  8. SET INT totalHoursPerYear = hrsPerPaycheck * paychecksPerYear
  9. SET REAL weeks = totalHoursPerYear/40

  11. END
